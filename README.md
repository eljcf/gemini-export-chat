# Gemini Chat Exporter

## 项目简介
 这是一个 Chrome 插件，旨在解决 LLM 对话的“记忆断层”问题。它可以将 Gemini 的完整对话记录导出为 Markdown/json 文件，方便在新对话中快速恢复上下文记忆。

## 核心功能

暴力回溯：自动向上滚动并加载所有懒加载的历史记录。

指纹去重：采用 V16.0 指纹识别算法，彻底消除对话重复抓取问题。

悬浮助手：在网页右下角提供实时进度显示与交互面板。

一键读档：内置专为 Gemini 优化的启动 Prompt，实现无缝记忆衔接。

## 安装教程 (开发者模式)

下载本项目并解压。

打开 Chrome，进入 chrome://extensions/。

开启“开发者模式”。

点击“加载已解压的扩展程序”，选择 src 目录。

刷新 Gemini 页面即可开始使用。
